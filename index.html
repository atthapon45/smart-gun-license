ขั้นตอน ลงทะเบียน ติดต่อ

ขั้นตอนการดำเนินการ

1. ลงทะเบียนและยืนยันตัวตน

ยืนยันด้วยบัตรประชาชนและ Face ID ผ่านระบบออนไลน์

2. ยื่นคำขอใบอนุญาต (ป.3)

กรอกข้อมูล เลือกประเภทปืน และแนบเอกสารผ่านพอร์ทัล

3. ตรวจสอบประวัติด้วย AI

เชื่อมข้อมูลกับตำรวจ, ศาล, สาธารณสุข ตรวจสอบแบบอัตโนมัติ

4. สอบทฤษฎีและปฏิบัติ

เรียนออนไลน์ + ทดสอบสนามยิงปืนที่ได้รับการรับรอง

5. ออกใบอนุญาต ป.3 (Digital)

ได้รับเอกสารผ่าน QR Code และเก็บใน Digital Wallet

6. เลือกร้านค้าและออก ป.4

ซื้อปืนที่อนุญาต เชื่อมหมายเลขปืนกับระบบกลาง

7. ติดตาม/ต่ออายุ/แจ้งเตือนอัจฉริยะ

ระบบ AI ตรวจพฤติกรรม และแจ้งเตือนความเสี่ยง

ลงทะเบียนยื่นขอใบอนุญาต (จำลอง)




ขั้นตอนทั้งหมด

  1. เข้าสู่ระบบ / ยืนยันตัวตนผ่าน ThaiD
  2. ยื่นคำขอใบอนุญาต (ป.3) + แนบเอกสาร
  3. ตรวจสอบประวัติอัตโนมัติ + ทดสอบความรู้
  4. ออกใบ ป.3 พร้อม QR Code
  5. เลือกซื้อปืนจากร้านที่ได้รับอนุญาต
  6. ออกใบ ป.4 และผูกข้อมูลปืนกับเจ้าของ
